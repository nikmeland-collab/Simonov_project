# Проект новостного портала "Фамилия_Студента News"

В рамках выполненной работы спроектировано программное обеспечение для отображения и управления новостным контентом в виде статического одностраничного веб-сайта. Разработанный новостной портал предназначен для представления актуальных новостей различных категорий с удобным пользовательским интерфейсом и интерактивными элементами.

На основе выполненного проекта, с помощью программных продуктов Visual Studio Code (или аналогичного текстового редактора), современных веб-браузеров (Chrome, Firefox, Safari) и технологий HTML5, CSS3, JavaScript (ES6+), DOM API, CSS Grid, Flexbox реализовано требуемое программное обеспечение.

Реализованное программное обеспечение позволяет:

- Отображать новостные карточки с изображениями, заголовками, датами публикации, категориями и анонсами
- Использовать семантическую HTML-разметку для улучшения доступности и SEO
- Применять адаптивный дизайн с использованием CSS Grid (3 колонки на широком экране, 2 на среднем, 1 на мобильном)
- Реализовывать плавные hover-эффекты при наведении на новостные карточки (изменение фона и тени)
- Хранить данные новостей непосредственно в HTML-разметке (15 новостных статей)
- Интерактивно показывать/скрывать анонсы новостей при клике на кнопку "Читать далее"/"Свернуть"
- Отображать топ-5 популярных новостей в боковой панели
- Навигацию по разделам сайта через меню в шапке страницы
- Корректно отображаться на различных устройствах благодаря адаптивной верстке

Созданное программное обеспечение опубликовано локально и может быть развернуто на любом веб-сервере или размещено на GitHub Pages для публичного доступа.

## Структура проекта

```
web_project/
├── index.html          # Главная HTML-страница (с встроенными новостями)
├── styles.css          # Файл стилей CSS
├── script.js           # JavaScript для интерактивности
├── content.txt         # Исходные данные новостей (15 записей)
├── src/
│   └── img.png         # Изображение-плейсхолдер для новостей
└── README.md           # Документация проекта
```

## Запуск проекта

### Простой способ (без сервера)
1. Откройте файл `index.html` в современном веб-браузере (Chrome, Firefox, Safari, Edge)
2. Все данные новостей встроены непосредственно в HTML, поэтому дополнительный веб-сервер не требуется
3. Для корректного отображения изображений убедитесь, что файл `src/img.png` находится в соответствующей папке

### Альтернативный способ (с локальным сервером)
Если требуется тестирование с локальным сервером (например, для проверки CORS или других функций):

**Python:**
```bash
python -m http.server 8000
```
Затем откройте в браузере: `http://localhost:8000`

**Node.js:**
```bash
npx http-server
```

**VS Code:**
Используйте расширение "Live Server" для автоматического запуска локального сервера

## Особенности реализации

### Интерактивность
- При загрузке страницы анонсы новостей скрыты по умолчанию
- При клике на кнопку "Читать далее" анонс плавно появляется с анимацией
- Текст кнопки меняется на "Свернуть"
- Повторный клик скрывает анонс и возвращает кнопку в исходное состояние

### Визуальные эффекты
- **Hover-эффекты**: При наведении курсора на новостную карточку:
  - Изменяется цвет фона
  - Увеличивается тень (box-shadow)
  - Карточка слегка поднимается (transform: translateY)
- **Плавные переходы**: Все изменения происходят с анимацией благодаря CSS transitions
- **Анимация появления**: Анонсы появляются с эффектом fadeIn (плавное появление с движением вверх)

### Дизайн-решения
- **Цветовая схема**: Использована темная шапка (#2c3e50) и светлый фон (#f4f4f4) для контраста
- **Типографика**: Четкая иерархия заголовков с различными размерами шрифтов
- **Карточки**: Скругленные углы (border-radius) и тени для создания эффекта глубины
- **Категории**: Цветные бейджи для визуального разделения новостей по категориям

## Технические детали

### Визуальный стиль
- Градиентный фон с несколькими radial/linear слоями
- Стеклянные карточки (glassmorphism): полупрозрачный фон, размытие, подсветка границ
- Современная типографика: шрифт Inter (Google Fonts)
- Акцентные градиенты для кнопок и бейджей категорий
- Легкие тени и плавные переходы для эффекта глубины
- Фиксированная шапка с блюром для улучшения навигации

### Используемые технологии и их назначение

#### HTML5
- **Семантические теги**: Использованы теги `<header>`, `<main>`, `<article>`, `<aside>`, `<footer>` для улучшения структуры документа, доступности и SEO-оптимизации
- **Мета-теги**: Настроены кодировка UTF-8 и viewport для корректного отображения на мобильных устройствах
- **Встроенный контент**: Все 15 новостных статей размещены непосредственно в HTML для быстрой загрузки без дополнительных запросов к серверу
- **Навигация**: Реализовано меню с якорными ссылками для навигации по разделам страницы

#### CSS3
- **CSS Grid Layout**: адаптивная сетка карточек (3 / 2 / 1 колонка)
- **Flexbox**: выравнивание элементов меню и мета-блока новостей
- **Градиенты и blur**: многоуровневый фон, стеклянные карточки, подсветка категорий и кнопок
- **CSS Анимации**: плавное появление анонсов (`fadeIn`) и hover-эффекты
- **Медиа-запросы (@media)**: breakpoints 640px и 968px
- **Псевдоклассы**: `:hover` для интерактивных эффектов
- **Box-shadow и transitions**: плавные переходы и эффект подъема карточек

#### JavaScript (ES6+)
- **DOM API**: Использован для манипуляции элементами страницы:
  - `document.querySelectorAll()` - для поиска всех кнопок
  - `document.getElementById()` - для доступа к элементам анонсов
  - `classList.add/remove()` - для управления классами CSS
- **Обработка событий**: Реализованы обработчики событий `click` для кнопок "Читать далее"
- **Event Listeners**: Использован `addEventListener` для привязки функций к событиям
- **DOMContentLoaded**: Применен для инициализации скриптов после полной загрузки DOM-структуры

### Структура HTML-документа

1. **`<header>`** - Шапка сайта:
   - Заголовок портала "Фамилия_Студента News"
   - Навигационное меню с тремя пунктами (Главная, Категории, Контакты)

2. **`<main>`** - Основной контент:
   - Секция с новостными карточками
   - Каждая карточка (`<article>`) содержит:
     - Изображение новости
     - Заголовок (`<h2>`)
     - Мета-информацию (дата и категория)
     - Скрытый анонс (по умолчанию)
     - Кнопку для показа/скрытия анонса

3. **`<aside>`** - Боковая панель:
   - Список топ-5 популярных новостей
   - Пронумерованный список с якорными ссылками

4. **`<footer>`** - Подвал сайта:
   - Информация о copyright
   - Контактный email

### Файлы проекта и их назначение

- **`index.html`** - Главный HTML-файл, содержащий всю структуру страницы и встроенный контент новостей. Является точкой входа в приложение.

- **`styles.css`** - Файл каскадных таблиц стилей, определяющий:
  - Визуальное оформление всех элементов
  - Адаптивную верстку для разных устройств
  - Анимации и переходы
  - Цветовую схему и типографику

- **`script.js`** - JavaScript-файл, реализующий:
  - Функционал показа/скрытия анонсов новостей
  - Обработку кликов по кнопкам
  - Инициализацию интерактивных элементов при загрузке страницы

- **`content.txt`** - Исходный файл с данными новостей в формате CSV (разделитель - точка с запятой). Использовался для первоначальной подготовки контента, который затем был встроен в HTML.

- **`src/img.png`** - Изображение-плейсхолдер, используемое для всех новостных карточек. Обеспечивает визуальную привлекательность и единообразие дизайна.

### Адаптивность и кроссбраузерность

- **Breakpoints**: 
  - Мобильные устройства: до 640px (1 колонка)
  - Планшеты: 641px - 968px (2 колонки)
  - Десктоп: от 969px (3 колонки)

- **Поддержка браузеров**: Проект использует стандартные веб-технологии, совместимые с современными браузерами (Chrome, Firefox, Safari, Edge)

### Контент

- **15 новостных статей** различных категорий:
  - Политика
  - Спорт
  - Технологии
  - Экономика
  - Культура
  - Наука
  - Общество
  - Здоровье
  - Образование
  - Туризм

Каждая новость содержит заголовок, дату публикации, категорию и развернутый анонс.
